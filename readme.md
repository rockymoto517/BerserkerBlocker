# Berserker Blocker

#### Blocks
- Maniac 
- Leech
- Crew Chief
- AI Boosts
- Combat Medic Aced

#### Other Features
- Allows you to disable them if you only have berserker, all the time, or never
- Auto-updating
- Support for individual profiles
